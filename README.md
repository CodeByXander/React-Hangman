# Hangman – React Capstone Project

A simple Hangman game built using React.  
This project demonstrates React components, event handling, state management, routing, and user interaction.

---

## How to Play

- The game picks a random secret word.
- Guess letters using the on-screen keyboard.
- Correct letters are revealed in the word.
- Wrong guesses increase the hangman drawing.
- If you reach 10 wrong guesses, you lose.
- If you reveal the whole word, you win.

You can request help from the “Help” menu.

---

## Getting Started

### 1. Clone or Download This Repository

git clone

### 2. Install Dependencies

npm install

### 3. Run the App

npm start

### 4. View app

go to http://localhost:3000/
